# 💸 가계부 웹 사이트

## 프로젝트 소개
> 수입과 지출을 기록하고, 차트로 소비 패턴을 분석하는 간단한 가계부 웹사이트

### 🔑 로그인
- ID와 비밀번호 입력 후 로그인하는 화면입니다.
- 회원가입을 하지 않았다면, 버튼을 클릭하여 가입할 수 있습니다.

(로그인 페이지 사진 넣기)
( + 회원가입 페이지 설명 및 사진)

### 📆 입금·출금 내역 확인
- 달력을 통해 날짜별 내역을 한눈에 볼 수 있습니다.
- 수입/지출 내역을 '추가, 조회, 수정, 삭제'할 수 있습니다.

(가계부 페이지 사진)

### 📊 통계
- 월간/주간 수입 및 지출 합계를 제공합니다.
- 카테고리별 지출 비중을 확인할 수 있습니다.

(통계 페이지 사진 넣기)
